# Security Policy

## Supported Versions

Only the latest `main` branch is supported.

## Reporting a Vulnerability

Please do not open a public issue for sensitive vulnerabilities.

Contact:
- Ralph Li ralphtaka@gmail.com
- https://www.linkedin.com/in/ralph-li-76749b6a

## Response Targets

- Acknowledgement: within 3 business days
- Initial triage: within 7 business days
- Mitigation / fix target: based on severity and exploitability

## Remediation SLA

Use the baseline in `security_docs/vulnerability-sla.md` for client projects unless contract terms override it.
